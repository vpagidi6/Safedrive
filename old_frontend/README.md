# Old Flutter Frontend

This folder contains the original Flutter/Dart mobile app frontend code that has been replaced by the Next.js web dashboard.

## Contents

- `lib/` - Dart source code
- `android/` - Android platform files
- `ios/` - iOS platform files
- `macos/` - macOS platform files
- `windows/` - Windows platform files
- `web/` - Flutter web platform files
- `assets/` - Images, icons, and other assets
- `test/` - Test files
- `build/` - Build artifacts
- `pubspec.yaml` - Flutter project configuration
- `pubspec.lock` - Dependency lock file

## Note

This code is kept for reference but is no longer actively used. The current frontend is in the `web-dashboard/` directory (Next.js).
